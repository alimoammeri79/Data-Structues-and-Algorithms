# Generatring Strings From Tokens
Given some string tokens and a constant `n`, This program generates all posssible concatinations of this tokens that are 
less or equal to `n` in size.

## Example

### Input
```
Tokens = {"00", "101", "111"}
n = 5
```

### Output
```
00
0000
00101
00111
101
10100
111
11100
```
